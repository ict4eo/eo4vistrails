DataAnalytics¶

This package provides statistical capabilities for eo4vistrails. R, SciPy are proposed to be used

GeoInf¶

This package provides GIS capabilities for eo4vistrails. In particular, provides GRASS, PostGIS, OGC clients and some visualisation capabilities through the QGIS API.

RPyC¶

This full package extends VisTrails, providing GIS/Earth Observation ingestion, pre-processing, transformation, analytic and visualisation capabilities . Included is the abilty to run code transparently in OpenNebula cloud environments.

Utils¶

This package provides core and general purpose utility code for eo4vistrails. In particular, provides a generic session object for connecting to cloud environments, GIS environments and statistical environments.
