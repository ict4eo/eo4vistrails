RPyC¶

This full package extends VisTrails, providing GIS/Earth Observation ingestion, pre-processing, transformation, analytic and visualisation capabilities . Included is the abilty to run code transparently in OpenNebula cloud environments.

RPyC¶

This module is the core; holding annotations and mixins.

class rpyc.RPyC.RPyCModule[source]¶

This forms the basis for ensuring that rpyc-based modules have the correct input ports.

class rpyc.RPyC.RPyCSafeMixin[source]¶

This module provides a developer with dummy versions of each of the methods that a module would normally supply.

An instance of a RPyC module created in VisTrails will have the proper vistrails module mixed in by the annotation. On a RPyC node the dummy verion is used to instantiate a shadow of the original module. The shadow’s methods are linked back to the original module.

class rpyc.RPyC.RPyCSafeModule(requiredVisPackages=[])[source]¶

RPyCSafeModule executes the compute method on a RPyC node.

This annotation mixes in the code needed to ensure both threadsafe execution and rpyc safe execution.
